# Java Data Structures Practice

Exercises and implementations of data structures and algorithms in Java.

Based on the book _“Estructuras de Datos en Java”_ by Luis Joyanes Aguilar & Ignacio Zahonero Martínez.

## Purpose

This repository is focused on learning and practicing data structures by implementing solutions manually, without using AI-generated code.

## Topics
- Big O Notation
- Linked Lists
- Stacks
- Queues
- Priority Queues
- Binary Trees
- AVL Trees
- Graphs
- Sorting Algorithms
- Searching Algorithms

## Technologies

- Java
- Visual Studio Code
- Git & GitHub
